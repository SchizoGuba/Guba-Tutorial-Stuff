To get ClientInstance::update first you need the ClientInstance vTable cause there are no strings you can search for in ClientInstance::update so we are gonna find ClientInstance::init first

To find ClientInstance::init you can search for the string '/ServerContent' Xref from it and you will land somewhere in ClientInstance::init the function definition should look something like

cpp
__int64 __fastcall ClientInstance::init(
        __int64 a1,
        __int64 a2,
        void (__fastcall ***a3)(_QWORD, __int64),
        __int64 a4,
        __int64 a5,
        __int64 a6,
        unsigned int a7,
        __int128 *a8)

Xref from ClientInstance::init and you should land in the ClientInstance vTable

Search through the vTable and you should find a function with a definition that looks like

cpp
__int64 __fastcall ClientInstance::update(__int64 a1, unsigned __int8 a2)
