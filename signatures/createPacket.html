The easiest way to find create packet is by searching 'UnknownPacket' in strings Check either one till you find 'aUnknownpacket_0' Select it and xref to the function it calls

Right above the string there should be a function that takes 2 arguments that should be your createPacket function

The function definition should look something like this

cpp
_QWORD *__fastcall MinecraftPackets::createPacket(_QWORD *a1, int a2)
